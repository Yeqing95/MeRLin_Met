# MeRLin-Met
MeRLin-Met (Melanoma Metastasis Lineage Tracing) is a high-resolution lineage tracing system that integrates single-cell RNA sequencing (scRNA-seq), RNA fluorescence in situ hybridization (RNA-FISH), and computational analyses to track the clonal dissemination and transcriptional evolution of melanoma cells from primary tumors to distant metastatic sites.

## Requirement packages
- R version 4.4.0
- Seurat version 5.3.0
- ClonoCluster version 0.0.1
- inferCNV version 1.20.0
- SCENIC version 1.3.1
- SCopeLoomR version 0.13.0
